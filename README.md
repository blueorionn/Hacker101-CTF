# Hacker101 CTF

The HackerOne CTF (often called **Hacker101** CTF) is a free, educational Capture The Flag platform designed to teach ethical hacking and web application security. It allows both beginners and experienced professionals to find simulated security vulnerabilities and earn "flags" in a safe, controlled environment.

This repo contains solutions for CTF levels [https://ctf.hacker101.com/ctf](https://ctf.hacker101.com/ctf).
